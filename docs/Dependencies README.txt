
	------------- PASO A PASO PARA EL FRONTEND Y API-------------  

Crear un nuevo venv en la carpeta "app"

1- Para eso hay que ejecutar:
	
	->python3 -m venv env
	->cd env/Scripts
	->activate

2- Ya que se logeó en el virtual environment, ejecutar el comando:
	
	-> pip install -r install.txt

3- Posteriormente, ejecutar en EL MISMO ORDEN:

	-> python3 qis.py
	-> python3 __init__.py

4- Abrir el FrontEnd (localhost:8080) y ejecutar la página web!
